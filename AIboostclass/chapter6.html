# Optuna

오픈소스 하이퍼 파라미터 튜닝 프레임워크

  • Eager search spaces

  • State-of-the-art algorithms

  • Easy parallelization

Sample

import optuna

def objective(trial):
    x = trial.suggest_uniform('x', -10, 10)
    return (x - 2) ** 2

study = optuna.created_study()
study.optimize(objecive, n_trials=100)

study.best_params
